## 🚀 Sobre o Projeto
O **FETEPS App** foi desenvolvido para modernizar a experiência da **Feira Tecnológica do Centro Paula Souza**, conectando visitantes, expositores, palestrantes e a equipe organizadora em um único aplicativo simples e intuitivo.

O app inclui:
- 🎨 Interface moderna com suporte a **tema claro e escuro**
- 🤖 Mascote interativo com frases dinâmicas
- 📍 Mapa da feira
- 🧠 Seção de curiosidades sobre os **ODS**
- 👥 Página completa sobre a equipe
- 📸 Avatares automáticos usando **DiceBear API**
- 🔐 Login, cadastro e tela inicial animada
- 📑 Diversas páginas institucionais integradas

---

## 🧩 Principais Funcionalidades
### 🎛️ Tema claro e escuro
Gerenciado dinamicamente via `ThemeProvider`, incluindo troca rápida pelo usuário.

### 🗺️ Mapa Interativo
Imagens e informações otimizadas para facilitar a navegação presencial na feira.

### 🧑‍🏫 ODS Explicados
Motivação dos projetos segundo a ONU, com expansão individual de cada um dos **17 Objetivos de Desenvolvimento Sustentável**.

### 🤝 Informações da Equipe
Apresentação organizada por setores:
- Orientadores
- Modelagem (Figma)
- Desenvolvimento Mobile
- Desenvolvimento Web
- Backend
- Ilustração

### 📲 Menu lateral completo
Navegação simples entre:
- Projetos
- Palestrantes
- Patrocinadores
- Sobre nós
- Participantes
- Curiosidades
- Perfil
